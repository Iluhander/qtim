Т.З. для QTIM

## Description

[Nest](https://github.com/nestjs/nest) CRUD сервис статей с аутентификацией.

## Поднятие зависимостей

```bash
$ docker-compose up -d
```

## Миграции

```bash
$ npm run migration:up
```

## Тесты

```bash
$ npm run test
```

## Запуск

```bash
$ npm run start:dev
```
